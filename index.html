Bootstrap-community

A template for starting a community website using the Bootstrap platform.

Download this project as a .zip file Download this project as a tar.gz file

What is Bootstrap-community?

Bootstrap-community is a template for use on startup websites where you want a beautiful page but you don't have the time and/or resources to create one.

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.

How to Setup

Bootstrap-community requires a few prerequsites before use.

Note: You may skip this if your host/server already includes these packages.

  • If your server runs a Debian-based distro (Debian/Ubuntu/Xubuntu/etc...)
    1. Install Apache: sudo apt-get install apache2
    2. Install MySQL: sudo apt-get install mysql-server mysql-client
    3. Install PHP5: sudo apt-get install php5 libapache2-mod-php5
    4. Restart Apache: sudo /etc/init.d/apache2 restart
    5. Install PHP5 Libraries: sudo apt-get install php5-mysql php5-curl php5-gd php5-idn php-pear php5-imagick php5-imap php5-mcrypt php5-memcache php5-mhash php5-ming php5-ps php5-pspell php5-recode php5-snmp php5-sqlite php5-tidy php5-xmlrpc php5-xsl php5-json
    6. Restart Apache (Again): sudo /etc/init.d/apache2 restart
    7. (Optional!) Install phpMyAdmin: sudo apt-get install phpmyadmin
    8. Download, extract, and upload bootstrap-community to your webroot folder (Default: /var/www
  • If your server runs a Windows OS
    1. Install WAMPServer or XAMPP.
    2. Download, extract, and upload bootstrap-community to your webroot folder (WAMP: C:\wamp\www; XAMPP: C:\xampp\htdocs

Rather Drive Stick?

If you prefer to not use the automatic generator, push a branch named gh-pages to your repository to create a page manually. In addition to supporting regular HTML content, GitHub Pages support Jekyll, a simple, blog aware static site generator written by our own Tom Preston-Werner. Jekyll makes it easy to create site-wide headers and footers without having to copy them across every page. It also offers intelligent blog support and other advanced templating features.
